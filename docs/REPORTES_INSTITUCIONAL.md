# Reportes Institucionales

Artefactos estándar:

- Playwright HTML: `playwright-report/`
- Allure: `allure-report/`
- Test results (json/junit): `test-results/`
- Performance k6 summary: `reportes/*.json`
- Security ZAP baseline: `reportes/security/`

Comandos útiles:

- `npm run report:html`
- `npm run report:allure:generate`
